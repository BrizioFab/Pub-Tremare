## 🎯 COLOR PSYCHOLOGY

### Navy (#1a3a52)
- **Meaning**: Trustworthiness, professionalism, stability
- **Usage**: Primary text, headings, buttons, backgrounds
- **Emotion**: Serious, elegant, nautical

### Teal (#2a5a72)
- **Meaning**: Calm, sophistication, marine essence
- **Usage**: Secondary buttons, borders, hover states
- **Emotion**: Fresh, contemporary, serene

### Gold (#d4a574)
- **Meaning**: Luxury, prestige, warmth, nautical heritage
- **Usage**: Accents, CTA buttons, highlighting important info
- **Emotion**: Premium, exclusive, inviting

### Cream (#f5f1e8)
- **Meaning**: Calm, neutral, sophisticated background
- **Usage**: Alternative backgrounds, card backgrounds
- **Emotion**: Approachable, warm, easy on the eyes

### WhatsApp Green (#25D366)
- **Meaning**: Action, trust, direct communication
- **Usage**: Primary CTA button for WhatsApp
- **Emotion**: Friendly, immediate, conversational

## 📝 TYPOGRAPHY SPECIFICATION

### Playfair Display (Headings)
```
h1: 2.5rem / 3rem (mobile/desktop)
h2: 2rem / 2.25rem
h3: 1.5rem
h4: 1.25rem
line-height: 1.2
letter-spacing: -0.5px to -1px (for larger sizes)
```

### Inter (Body)
```
body: 1rem (16px)
line-height: 1.6
letter-spacing: normal
font-weight: 400 (regular) for body, 600 for emphasis
```

### Hierarchy
1. Main headings (h1): Hero title, page titles
2. Section headings (h2): Featured menu, how it works
3. Card headings (h3): Menu items, contact cards
4. Labels (h4): Small section labels

## 🎓 IMPLEMENTATION NOTES

### For Developers
1. **CSS**: Mobile-first is enforced. Start with base styles, then add media queries.
2. **HTML**: Use semantic tags (section, article, nav, header, footer).
3. **JavaScript**: Keep it simple. Only use vanilla JS for navbar toggle and smooth scrolling.
4. **Images**: Replace placeholder descriptions with actual image files.
5. **Fonts**: Playfair Display and Inter load from Google Fonts (add to base.html <head>).

### For Designers
1. Keep the navy/gold/teal color scheme consistent.
2. Icon usage: Emojis for simplicity, SVG for quality.
3. Whitespace: Generous spacing for premium feel.
4. Imagery: High-quality, professionally shot (not stock-obvious).
5. Typography: Large headlines, comfortable reading size.

### For Content Writers
1. Keep copy short and scannable.
2. Use active voice.
3. Lead with benefits, not features.
4. Always include a clear CTA.
5. Proofread in Italian and English.
